This is the embedded code used for data collection in the WIHMS garment as of 7/5/2020.

Relies on the following libraries - 
Adafruit Sensor Library: https://github.com/adafruit/Adafruit_Sensor
Adafruit BNO055: https://github.com/adafruit/Adafruit_BNO055
Adafruit MCP9808 Library: https://github.com/adafruit/Adafruit_MCP9808_Library
Sparkfun MAX3010X Sensor Library: https://github.com/sparkfun/SparkFun_MAX3010x_Sensor_Library
Bill Greiman's SSD1306Ascii: https://github.com/greiman/SSD1306Ascii
Bill Greiman's SdFat Library: https://github.com/greiman/SdFat
Andrew Wickert's DS3231 Library: https://www.arduinolibraries.info/libraries/ds3231
